FreeRDP 远程连接 windows 桌面

发表于 2018-01-08

安装 FreeRDP

  • CentOS 安装 FreeRDP
  • 安装中如出现找不到的包的情况请替换源，建议使用阿里源或网易源，其网站首页均有使用帮助。

使用 FreeRDP

  • xfreerdp 命令参数较多，可编写如下脚本使用。

  • 参数说明

    -f: 全屏显示，切换全屏快捷键为 ctrl + alt + enter
    -u: 用户名
    -p: 密码
    -g: 窗口大小
    -a: 32位颜色
    -x: lan 局域网模式
    –sec nla: 网络级别身份验证
    –plugin rdpsnd –data pulse –: 远程声音播放
    –plugin cliprdr: 共享剪贴板
