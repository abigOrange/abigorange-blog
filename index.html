锐化扫描版pdf

扫描版pdf的文字多数存模糊的情况，很影响阅读体验，以下通过软件通过锐化pdf的方法。

使用软件

  • 1.Solid PDF Tools
  • 2.Photoshop

具体步骤

1.提取原始图像

  • 使用Solid打开pdf文件，提取原始图像

2.录制动作

  • Photoshop打开一张提取的图像，点击窗口->动作(alt+F9)->创建新动作->记录

3.锐化图片

  • 点击滤镜->锐化->智能锐化，调整半径数值越大锐化越重
  • 调整好后，点击文件->存储为(shift+ctrl+s)，新建文件夹保存改图像
  • 停止动作录制
  • 保障后面的批处理，删除刚保存的图像

4.批量处理所有图像

  • 点击文件->自动->批处理
  • 选择原始图像所在路径
  • 选择刚录制的动作
  • 点击确定，等待批量处理完成

5.合并成pdf

  • Solid 中点击文件->创建或合并pdf
  • 添加批量处理的图像
  • 点击创建，生成pdf
  • 保存生成的pdf

关闭 Gnome Hot Corner

发表于 2018-01-17

Centos7 下鼠标移至左上角总弹出概览界面，甚是烦人。google一通乱搜后竟毫无结果，最后转变思路以 Gnome 为关键字搜索，终皇天不负有心人，下面是解决方法。

No Topleft Hot Corner

  • 直接在 Application Installer 中 搜索 No Topleft Hot Corner 并下载。
  • 也可以上 Gnome extension 网站下载。
  • 安装好后在优化工具 扩展中打开。

FreeRDP 远程连接 windows 桌面

发表于 2018-01-08

安装 FreeRDP

  • CentOS 安装 FreeRDP
  • 安装中如出现找不到的包的情况请替换源，建议使用阿里源或网易源，其网站首页均有使用帮助。

使用 FreeRDP

  • xfreerdp 命令参数较多，可编写如下脚本使用。

  • 参数说明

    -f: 全屏显示，切换全屏快捷键为 ctrl + alt + enter
    -u: 用户名
    -p: 密码
    -g: 窗口大小
    -a: 32位颜色
    -x: lan 局域网模式
    –sec nla: 网络级别身份验证
    –plugin rdpsnd –data pulse –: 远程声音播放
    –plugin cliprdr: 共享剪贴板
